# Application Features

Dash Bash Utility is packed with features designed for speed and efficiency during active delivery shifts.

## 1. Target Calculator
The Target Calculator helps drivers hit specific financial goals ($99, $120, or custom) by calculating optimal combinations of delivery amounts.
- **Dynamic Calculation**: Updates instantly as you add or remove amounts.
- **Color Coding**: Results are highlighted in green when optimal and red when far from the target.
- **Goal Setting**: Quickly toggle between standard targets or input a custom amount.

## 2. Dasher Management
A comprehensive system for tracking multiple delivery accounts ("Dashers").
- **24-Hour Timers**: High-precision countdowns (HH:mm:ss) for tracking account usage or cooling periods.
- **Status Badges**: Visual indicators for Crimson, FastPay, Red Card, and Appealed statuses.
- **Earnings Tracking**: Record earnings directly within the dasher card.
- **Cash-Out History**: Track every withdrawal with method, amount, and automatic timestamping.
- **Category Organization**: Group dashers into custom categories with drag-and-drop support.

## 3. Address Book
Store frequently visited or problematic locations with intelligence.
- **Operating Hours**: Enter store hours to see a live "Open/Closed" status.
- **Time Remaining**: Color-coded indicators show how much time is left before a store closes (Red < 60m, Yellow < 120m, Green > 120m).
- **Categorization**: Group stores by zone, type, or preference.
- **Quick Copy**: One-click address copying for easy navigation.

## 4. Quick Messages
Speed up communication with customers using pre-written templates.
- **Template Management**: Create and edit message templates.
- **Drag-and-Drop**: Reorder messages based on frequency of use.
- **Copy-to-Clipboard**: One-tap copy with visual success confirmation.

## 5. Notes System
A flexible note-taking tool for miscellaneous information.
- **Categorized Notes**: Keep different types of information separate.
- **Resizable Fields**: Adjust note height for longer entries.
- **Drag-and-Drop**: Move notes between categories easily.

## 6. Statistics Dashboard
Real-time analytics for your delivery business.
- **Active Dashers**: Counts per category.
- **Financial Summary**: Total balances across all accounts.
- **Status Breakdown**: See how many Red Card holders or Crimson dashers you have at a glance.
